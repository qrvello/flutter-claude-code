# Flutter Backend Integration

Agents for integrating various backend services and APIs.

## Included Agents

### flutter-rest-api
Specializes in integrating REST APIs with Flutter using Dio and HTTP clients. Covers JSON serialization, error handling, and API service architecture.

### flutter-graphql
Specializes in integrating GraphQL APIs with Flutter using graphql_flutter. Covers query/mutation/subscription patterns, caching, and real-time data.

### flutter-firebase
Specializes in integrating Firebase services including FlutterFire, Firebase Authentication, Cloud Firestore, Cloud Storage, Cloud Messaging (FCM), and Firebase Analytics.

### flutter-aws
Specializes in integrating AWS services including AWS Amplify, Cognito authentication, API Gateway, S3 storage, Lambda functions, and AppSync.

## Installation

```bash
/plugin marketplace add /path/to/flutter-claude-code
/plugin install flutter-backend@flutter-claude-code
```

## Example Usage

```
Integrate our REST API with JWT authentication and proper error handling

Set up Firebase authentication with email and Google sign-in

Connect my Flutter app to AWS AppSync GraphQL API
```
