# Flutter Architecture & State Management

Agents for application architecture and state management patterns.

## Included Agents

### flutter-bloc
Specializes in implementing BLoC pattern state management. Covers the complete BLoC ecosystem: bloc, flutter_bloc widgets, bloc_test, hydrated_bloc persistence, replay_bloc undo/redo, and bloc_concurrency event transformers.

### flutter-architect
Designs Flutter application architecture and project structure. Specializes in Clean Architecture, feature organization, dependency injection, navigation, and scalable code organization.

## Installation

```bash
/plugin marketplace add /path/to/flutter-claude-code
/plugin install flutter-architecture@flutter-claude-code
```

## Example Usage

```
Set up a new e-commerce Flutter app with Clean Architecture

Implement BLoC pattern for authentication with login, logout, and session management
```
