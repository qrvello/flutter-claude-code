# Flutter Deployment

Agents for deploying Flutter apps to iOS App Store and Google Play Store.

## Included Agents

### flutter-ios-deployment
Specializes in deploying Flutter apps to the Apple App Store. Covers iOS app signing, TestFlight, App Store Connect, provisioning profiles, and release management.

### flutter-android-deployment
Specializes in deploying Flutter apps to the Google Play Store. Covers Android app signing, Google Play Console, Play Store optimization, and release management including staged rollout.

## Installation

```bash
/plugin marketplace add /path/to/flutter-claude-code
/plugin install flutter-deployment@flutter-claude-code
```

## Example Usage

```
Help me deploy my Flutter app to the App Store with proper signing and TestFlight beta testing

Guide me through deploying to Google Play Store with staged rollout
```

## Prerequisites

- **iOS**: Apple Developer account, certificates, provisioning profiles
- **Android**: Google Play Console account, app signing keys
