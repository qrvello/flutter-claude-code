# Flutter Platform Integration

Agents for iOS and Android native integrations and device management.

## Included Agents

### flutter-ios-integration
Specializes in integrating iOS-specific features, native code, and platform APIs. Covers Swift/Objective-C integration, platform channels, iOS frameworks, and Xcode configuration.

### flutter-android-integration
Specializes in integrating Android-specific features, native code, and platform APIs. Covers Kotlin/Java integration, platform channels, Android frameworks, and Gradle configuration.

### flutter-platform-channel-architect
Designs cross-platform communication architecture between Flutter and native code. Specializes in platform channel patterns, type-safe interfaces, and coordinating iOS/Android implementations.

### flutter-device-orchestrator
Manages iOS simulators and Android emulators for Flutter development. Specializes in device launch, app installation, screenshot capture, and multi-device testing.

## Installation

```bash
/plugin marketplace add /path/to/flutter-claude-code
/plugin install flutter-platform@flutter-claude-code
```

## Example Usage

```
Integrate HealthKit data into my Flutter app

Implement Android WorkManager for background tasks

Launch an iPhone 15 Pro simulator and run my Flutter app on it
```
