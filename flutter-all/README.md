# Flutter All - Complete Development Suite

This plugin includes all 19 Flutter development agents and the comprehensive patterns skill.

## What's Included

### UI & Design (4 agents)
- flutter-ui-designer
- flutter-ui-implementer
- flutter-ui-comparison
- flutter-design-iteration-coordinator

### Testing & Performance (3 agents)
- flutter-testing
- flutter-performance-analyzer
- flutter-performance-optimizer

### Architecture & State (2 agents)
- flutter-state-management
- flutter-architect

### Platform Integration (4 agents)
- flutter-ios-integration
- flutter-android-integration
- flutter-platform-channel-architect
- flutter-device-orchestrator

### Backend Integration (4 agents)
- flutter-rest-api
- flutter-graphql
- flutter-firebase
- flutter-aws

### Deployment (2 agents)
- flutter-ios-deployment
- flutter-android-deployment

### Skills (1 skill)
- flutter-patterns (comprehensive patterns for widgets, testing, performance, security, and animations)

## Installation

```bash
/plugin marketplace add /path/to/flutter-claude-code
/plugin install flutter-all@flutter-claude-code
```

## Usage

After installation, agents are automatically available. Simply describe your Flutter development task and Claude Code will use the appropriate agent(s).

For the patterns skill:
```
/flutter-patterns
```

See [PLUGIN_INSTALLATION.md](../PLUGIN_INSTALLATION.md) for more details.
