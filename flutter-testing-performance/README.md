# Flutter Testing & Performance

Agents for comprehensive testing and performance optimization.

## Included Agents

### flutter-testing
Specializes in writing tests for Flutter applications including unit tests, widget tests, integration tests, BLoC testing, mocking, and test-driven development.

### flutter-performance-analyzer
Profiles and analyzes Flutter app performance using DevTools. Specializes in identifying jank, memory leaks, and performance bottlenecks.

### flutter-performance-optimizer
Implements performance optimizations in Flutter apps. Specializes in widget optimization, efficient rendering patterns, and applying best practices like const constructors and keys.

## Installation

```bash
/plugin marketplace add /path/to/flutter-claude-code
/plugin install flutter-testing-performance@flutter-claude-code
```

## Example Usage

```
Write comprehensive tests for my authentication feature

My Flutter app is laggy when scrolling. Help me find and fix the bottleneck
```
